# Game QA Bug Reports in Hytale

This repository contains structured bug reports created during independent game testing of pre-release builds.

Each folder represents a single reported issue and includes:
- Clear reproduction steps
- Expected vs actual behavior
- Severity assessment
- QA notes and context

Focus areas:
- Gameplay systems
- Physics and entity behavior
- World interaction and edge cases

Reported by: Krimm (@Krimm_QA)
